# VS Code Environment

Минималистичная и автоматизированная настройка Visual Studio Code. Этот репозиторий содержит PowerShell-скрипт (`install.ps1`) для быстрой и надежной настройки рабочего окружения с нуля.

## Особенности

- **Автоматическая установка**: Скрипт самостоятельно скачивает и настраивает все необходимое напрямую из GitHub.
- **Пользовательская тема**: Установка кастомной темы для VS Code напрямую из релизов [vscode-theme](https://github.com/teenageswag/vscode-theme).
- **Улучшенные комментарии**: Установка расширения для продвинутой подсветки комментариев напрямую из релизов [vscode-better-comments](https://github.com/teenageswag/vscode-better-comments).
- **Типографика**: Автоматическое скачивание, установка и регистрация в системе шрифта Geist Mono.
- **Готовые настройки**: Загрузка актуального `settings.json` и автоматическая установка нужных публичных расширений из `extensions.txt` если это необходимо.
- **Быстродействие**: Установка происходит в разы быстрее, так как скрипт скачивает уже скомпилированные `.vsix` файлы из GitHub Releases и больше не требует локальной сборки через npm.

## Требования

Перед запуском убедитесь, что в вашей системе установлены и доступны в PATH:

- [Visual Studio Code](https://code.visualstudio.com/) (с включенным `code` в CLI)
- *Git и Node.js больше не требуются для работы скрипта!*

## Установка

Запустите скрипт через PowerShell (права администратора **не требуются**):

```powershell
.\install.ps1
```

*(Или кликните правой кнопкой мыши по `install.ps1` и выберите **Выполнить с помощью PowerShell**).*

> **Примечание:** Скрипт скачает нужный шрифт, расширения из GitHub Releases, применит пользовательские настройки и установит нужные публичные расширения без ручного вмешательства. В случае ошибки (например, проблем с сетью или отсутствия `code` в PATH), выполнение будет корректно прервано с выводом информативного сообщения, а все временные файлы удалены.
